Z80 Opcode Tables

Referencia visual de la CPU Z80 usada per Sega Master System, amb les dades separades del render i carregades des de JSON per blocs.

Bloc

Com llegir la taula

Cada bloc es mostra en una graella de 16x16. La fila indica el nibble alt del byte final i la columna indica el nibble baix. En blocs prefixats com DD CB d xx o FD CB d xx, la graella es basa en l'ultim byte xx, mentre que d es el desplacament signat respecte a IX o IY.

Llegenda

  • Clck = cicles o T-states de la instruccio.
  • Siz = mida total de la instruccio en bytes.
  • Op-Code = bytes exactes que l'assembler o la CPU veuen.
  • Description = explicacio simplificada del comportament.
